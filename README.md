# Simulation

Пошаговая симуляция 2D мира, населённого травоядными и хищниками.

## О проекте

Мир представляет собой матрицу NxM. Каждую клетку занимает один объект или существо. Существа двигаются, ищут еду и умирают от голода. Популяции пополняются автоматически.

**Объекты мира:**
- 🐇 Травоядные — ищут траву, гибнут от голода и хищников
- 🐺 Хищники — охотятся на травоядных, гибнут от голода
- 🌿 Трава — ресурс для травоядных, периодически восстанавливается
- 🗿 Камни — статичные объекты, занимают место

Поиск пути реализован через алгоритм BFS (поиск в ширину).

При запуске вводится размер карты от 5x5 до 20x20. Пробел — пауза/продолжение симуляции.

Необходимо установить зависимость из requiremets.txt для работы симуляции, ну или просто скачать "readchar" т.к. на нём завязана пауза симуляции.
